
Обновление конфигурации коммутатора asw1

Установить программу tftpd32.
Скопировать в папку с программой файл asw1.conf
Запустить tftpd32.exe
Залогиниться на коммутатор по SSH и выполнить следующие команды строго по порядку:
	conf t
	sdm prefer lanbase-default
	write memory
	copy tftp://x.x.x.x/asw1.conf start
где x.x.x.x - IP address компьютера с установленной программой tftpd32.

Затем перезагрузить по питанию
